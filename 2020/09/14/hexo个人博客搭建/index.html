你需要的

  1. 安装git
  2. 安装Node.js
  3. 安装Hexo
  4. 在github上创建个人仓库并部署博客

1.安装git

windows：到git官网上下载,Download git,下载后会有一个Git Bash的命令行工具，以后就用这个工具来使用git。安装好后记得使用 git --version 来查看版本并确认你已经成功安装。

以windows为例 使用win+r并输入cmd打开命令行再使用git --version即可

此后命令行可被代替为git bash，在git bash中输入命令更为方便

2.安装Node.js

前往node.js官网下载选择LTS版本并安装即可。

安装完成后同样使用命令查看是否成功安装
